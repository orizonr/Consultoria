ℹ️ Sobre o armazenamento de dados
O pipeline, agenda, log e configurações são armazenados no localStorage do navegador. Os candidatos e vagas ficam no Supabase (banco de dados na nuvem) — esses dados persistem independentemente do navegador. O backup completo cobre apenas os dados locais.
